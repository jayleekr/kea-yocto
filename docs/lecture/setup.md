# Yocto 빌드 환경 설정

!!! tip "Docker 기반 환경"
    이 강의에서는 Docker를 사용하여 일관된 개발 환경을 제공합니다.

## 빠른 시작

```bash
# 프로젝트 클론
git clone https://github.com/jayleekr/kea-yocto.git
cd kea-yocto

# Docker 환경 시작
./scripts/quick-start.sh
```
